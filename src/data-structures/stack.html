What is Stack?
Stack is a LIFO(Last in First out) structure or we can say FILO(First in Last out). push() function is used to insert new elements into the Stack and pop() function is used to remove an element from the stack.

Usage: We will add/remove the items in stack using push/pop. Best basic example is to check palindrome word. This works exactly with a stack concept. For more details, check source code/debug while it runs.
