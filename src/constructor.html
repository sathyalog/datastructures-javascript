What is a constructor?
Constructor function allows to create multiple objects of same flavor. Eg: Facebook has 1M+ users in global.Creating a new user required Name,Age,Gender,Email etc which are all properties. Hence we will be creating `n` user objects with one class using a User constructor function.

What is a prototype?
Prototype - prototype object is a simply an object that multiple other objects can refer to, To get any info/functionality that they need. For our needs each of our constructor functions have a prototype that all other instances will be able to refer to. Eg: If i need to add a property like emailaddress as facebook.com to all users, i can do it through prototype. Similarly, if we need to generate a new facebook email id to all users, we can achieve it using prototype methods. For more, please check the below link.
